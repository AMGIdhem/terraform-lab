# Terraform Lab - Oracle Cloud Infrastructure (OCI)

This repository contains Terraform configurations and learning exercises for provisioning infrastructure on Oracle Cloud Infrastructure (OCI).

## 📌 Overview

The goal of this project is to practice Infrastructure as Code (IaC) using Terraform with OCI. It includes basic examples for deploying and managing cloud resources in a reproducible and automated way.

## 🛠️ Technologies Used

- Terraform
- Oracle Cloud Infrastructure (OCI)
- HCL (HashiCorp Configuration Language)
